# Words Extraction

Скрипты извлекает редкие слова из книг (fb2 или epub) и собирает колоды Anki.  
    
\* Cкрипт откидывает слова из словаря Oxford 3000 words, под этот словарь сделан отдельный deck.

## Установка

```bash
make install
```

## Настройка
Положите книгу в `data/books`

В `src/config.py` сконфигурируйте 2 переменные:
- `BOOK_NAME` — имя файла книги из `data/books` (с расширением) 
- `DECK_NAME` — имя колоды Anki

## Перевод 

Перевод выполняется через api гугл переводчика - библиотека `deep-translator`.

## Поиск редких слов
Cоберет csv из 3 полей - англ слово, перевод, кол-во упоминаний в тексте
Результат сохраняется в `output/handled_books/<имя_книги>_counter.csv`.

```bash
make run-main
```

## Сборка Anki Deck
Соберет базовую колоду Anki.
Результат сохраняется в `output/decks/<имя_книги>.apkg`.

```bash
make run-anki
```

## Стандартный алгоритам 
- положить книгу в `data/books` 
- сконифигурировать `BOOK_NAME` и `DECK_NAME`
- выполнить `make run-main` - соберется csv
- выполнить `make run-anki` - соберется колода
